hydration-next

Hydration Next

https://www.npmjs.com/package/hydration-next Simple library for serializing and rehydrating JavaScript objects with types memory. Designed as a drop-in replacement for JSON.parse and JSON.stringify

Installation

 
npm install hydration-next

or

yarn add hydration-next
